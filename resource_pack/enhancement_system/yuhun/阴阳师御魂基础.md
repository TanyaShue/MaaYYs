# 阴阳师 · 御魂基础备忘

本文档收录《阴阳师》中与普通御魂、首领御魂相关的基础设定，便于后续查阅。

---

## 一、主属性与各号位副属性（普通御魂 · 首领御魂相同）

以下规则对**普通御魂、首领御魂一并适用**；首领御魂**无另行例外**。

### 1\. 主属性（主词条）

每个位置仅能出现对应规则下的主属性。

| 位置 | 主属性 |
| --- | --- |
| **1 号位** | **固定**：固定数值的攻击加成 |
| **2 号位** | **随机**：攻击加成%、防御加成%、生命加成%，或 **固定数值**的速度加成 |
| **3 号位** | **固定**：固定数值的防御加成 |
| **4 号位** | **随机**：攻击加成%、防御加成%、生命加成%、效果命中%、效果抵抗% |
| **5 号位** | **固定**：固定数值的生命加成 |
| **6 号位** | **随机**：攻击加成%、防御加成%、生命加成%、暴击%、暴击伤害% |

**主词条上限（约指强化满级 +15）**

| 主属性名称 | 主属性上限值 |
| --- | --- |
| 暴击 | 55% |
| 攻击加成 | 55% |
| 速度 | 57 |
| 暴击伤害 | 89% |
| 生命加成 | 55% |
| 攻击 | 486 |
| 防御 | 104 |
| 生命 | 2052 |

简要归纳：**1 / 3 / 5** 号为**固定数值**攻防血；**2 / 4 / 6** 号为随机池（其中 **2 号位**是唯一会出现「固定数值速度」主属性的位置）。

### 2\. 副属性 · 词条条数（腿）、取值区间与强化

掉落时御魂可同时带有 **2、3、4 条初始副属性**（约定俗称作 **2 / 3 / 4 腿**）。

**副属性条目与单次数值范围**

下文 **11** 种副属性的「一次成长」可用的**最小值、最大值**。**所有情境统一**：\*\*+0（未强化）**已经 roll 到的初值；瘸腿御魂强化时「**多长一条新副属性**」给它的**第一段数值**；或任意阶段「**加厚某条既有副属性**」增加的那一档；乃至于 四条腿齐备后被选中的线在当次拿到的增量——上述每一次实际变动，都只对**本条所属类型**，在表中对应行的** `**[最小值，最大值]**` **内取样，取样方式一概为**均匀分布\*\*。

| 类型 | 最小值 | 最大值 |
| --- | --- | --- |
| 攻击 | 21.6 | 27 |
| 生命 | 91.2 | 114 |
| 防御 | 4 | 5 |
| 防御加成 | 2.4% | 3% |
| 暴击 | 2.4% | 3% |
| 攻击加成 | 2.4% | 3% |
| 生命加成 | 2.4% | 3% |
| 速度 | 2.4 | 3 |
| 效果抵抗 | 3.2% | 4% |
| 效果命中 | 3.2% | 4% |
| 暴击伤害 | 3.2% | 4% |

> 说明：除「速度」及固定点数攻防血外，上表中带 **%** 的条目与游戏内百分比副属性一致。

与各号位、主词条的互斥等池规则从略。

**强化与副属性成长**

*   御魂最高 **+15** 级；自 **+3** 起，每再 **+3** 级（即 **+3 / +6 / +9 / +12 / +15**）会触发 **1 次**副属性变动，**加满共 5 次**。
*   **2 / 3 腿**（俗称瘸腿，副属性条目不满 **4** 条）：在强化过程中只要把副属性「**补出一条新的**」（相对原有条目多出一类），新增的**那条类型**永远在全部 **11** 种副属性（见上表）上做**均匀抽签**，某一种被抽中的概率均为 **1/11 ≈ 9.0909091%；这一过程持续到条目加满至 4 条**。（本条只描述「抽到哪一种」；抽到之后该副属性拿到的**数值**，与所有其它情形一样，已由上一段统一说明。）
*   **4 腿**：每次触发都从「当前每条副属性各占一条腿的权重」中抽取要强化的那一条（**本节只刻画「抽到谁」**，命中后该条线当次的数值取样仍由上段「单次数值范围」统一给定）：
    *   第 **1** 次：4 条线各 **1 / 4**。
    *   第 **2** 次：**把上一轮被点到的那条线，再算作多一条虚拟腿**：此时等价于 **5 条虚拟腿各占 1 / 5**（被强化过的那条在虚拟池里重复出现，故实际概率更高）。
    *   依此类推：某条属性**被强化的次数越多**，在下一次抽取里占的虚拟份数越多，**后续再被点中的概率就越高**。（一般地，每多命中某条一次，它在下一轮「虚拟腿池」里就多占一格，与尚未被命中的线相比权重持续拉高。）

## 三、首领御魂

与普通御魂并列的一类御魂；**在同一号位上，主属性、副属性规则见第一节**。

*   **固有属性**：每件随机落在下列**加成类型**之一；名目与第二节纯数值两件套的「加成类型」**一致对齐**，唯独**不出现暴击伤害**。

| 加成类型（固有） | 固有数值 |
| --- | --- |
| 攻击 | +8% |
| 暴击 | +8% |
| 生命加成 | +8% |
| 效果命中 | +8% |
| 效果抵抗 | +8% |
| 防御加成 | +16% |

*   **两件套**：为**套装特殊效果**（战斗机制向描述），**不是**普通御魂那种两件套「直接叠加面板百分比」的写法。
