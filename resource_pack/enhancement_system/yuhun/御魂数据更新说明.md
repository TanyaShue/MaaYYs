## 二、关键概念说明

### 1. 【极限】的含义
- **定义**：指强化时，每次副属性的增长都加在了同一条属性上（低练度歪1-2次也可考虑留）
- **属性对应**：
  - `极限速度` → 速度属性（如速度 +16.61）
  - `极限生命/攻击等` → 对应带%的属性（如生命加成 16%，攻击加成 16%）
  - **注意**：纯数字的属性（如攻击 +100）不算极限

### 2. 2号位副属性特殊说明
- **速度拉满**：
  - 主属性速度 + 副属性速度拉满 = 俗称的"头"
  - 主属性速度 + 副属性速度收益差1~2次，练度低也可以留
  - 主属性不是速度，但副速度拉满，可以考虑留（如主攻击加成、副速度拉满的狂骨，可用于魂土反堆速度）

### 3. 路线添加规则
根据主属性和副属性的组合，决定可添加的路线：

| 主属性 | 副属性包含 | 可添加路线 |
|--------|-----------|-----------|
| 攻击加成 | 暴击/暴伤 | 常规输出 |
| 生命加成 | 暴击+暴伤 | 治疗量 |
| 防御加成 | 暴击/暴伤 | 防御输出 |
| 暴击 | 任意 | 常规输出、生命加成、防御输出 |
| 暴伤 | 任意 | 常规输出、生命加成、防御输出 |

## 六、常见问题

### Q1: 如何确定一个御魂有哪些路线？
**A**: 根据表格中"推荐副属性"列：
- 包含"暴击"和"暴击伤害" → 有"常规输出"路线
- 包含"暴击+暴伤"且主属性是生命加成 → 有"治疗量"路线
- 包含"极限速度" → 有"极限速度"路线
- 以此类推

### Q2: "暴击+暴伤"是什么意思？
**A**: 表示同时需要暴击和暴击伤害两个属性，通常简写为"暴击+暴伤"。在配置中需要展开为：
```json
["暴击", "暴击伤害"]
```

### Q3: 多条副属性如何排序？
**A**: 按优先级排序：
1. 核心属性（如暴击、暴伤）
2. 极限属性（如极限速度）
3. 其他有效属性

### Q4: 如何处理"无"的情况？
**A**: 如果某主属性表格标注"无"，表示该主属性此御魂一般不保留，可以不配置该条目或配置为空数组。
